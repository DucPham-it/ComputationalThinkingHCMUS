# Feature input/output checklist

## Recommendation input
- free-text query
- destination
- entertainment type
- budget
- time
- companion type
- browser GPS
- manual address
- favorite history
- rating preferences

## Recommendation output
- name
- place type
- address
- distance
- average rating
- price level
- open status
- photo
- detail action
- explanation score

## Detail output
- full address
- description
- rating and review count
- images
- opening hours
- price level
- contact info
- review list

## Route output
- origin
- destination
- distance
- duration
- polyline
- instructions

## Failure cases
- no matching places
- browser location unavailable
- place unavailable
- map data failed to load
